About Me

I started my career building the invisible backbone of software — the APIs, data layers, and systems that quietly keep everything running.

For years, C# and .NET were my tools of choice. I became fluent in the precision of backend engineering: designing database schemas, architecting REST services, and solving the kinds of problems that never show up in a UI but matter most when they break.

Now, I'm channeling that same engineering rigor into a new frontier: agentic AI. I'm learning Microsoft's Semantic Kernel framework and Anthropic's Claude Code to build systems that don't just execute instructions — they reason, plan, and act. The shift isn't just about new tools. It's about a new way of thinking about what software can do.
